# LUX Protocol: The Light of Science 💎🔬

**LUX** is a decentralized computing protocol designed to transform digital energy into scientific progress. Unlike traditional cryptocurrencies that consume resources for arbitrary calculations, LUX utilizes **Proof of Scientific Contribution (PoSC)** to power global research in medicine, astronomy, and advanced physics.

## 🌟 Vision
"From the Darkness of Speculation to the Light of Science."
LUX aims to build the world's largest decentralized super-computer, rewarding users for contributing their processing power to solve humanity's greatest challenges.

- **Ticker:** $LUX
- **Total Supply:** 100,000,000 LUX (Fixed)
- **Architect Allocation:** 5,000,000 LUX (Genesis Founder Vault)
- **Scientific Mining Reserve:** 95,000,000 LUX (Distributed via Contribution)

## 🛠 Technical Architecture
The LUX Protocol is built on the **Ethereum Blockchain (ERC-20)**, ensuring maximum security and transparency. 

### Core Features:
1. **PoSC Algorithm:** Validates computational tasks related to protein folding, genomic sequencing, and deep space data analysis.
2. **Founder Vault:** A time-locked allocation for the **Architect** to ensure long-term project stability.
3. **Open Source:** Fully auditable smart contracts powered by OpenZeppelin.
